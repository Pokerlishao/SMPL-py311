 # SMPL Based on Python3.11

This repo is able to load both v1.0.0 and v1.1.0 SMPL models. And you can download the SMPL models in [here](https://smpl.is.tue.mpg.de/).

This repo only relies on the following:
 - Python = 3.11
 - scipy = 1.11.1
 - numpy = 1.25.2

 ## RUN
    python main.py
